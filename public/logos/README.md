# Company Logos

Place your company logo files here.

## Required Logo Files:
- wpengine.svg
- google-cloud.svg
- gsv-ventures.svg
- pepsico.svg
- staq.svg
- bank-of-ethio.svg

## Image Specifications:
- Format: SVG (preferred) or PNG with transparent background
- Color: White or light gray (will be displayed on dark background)
- Size: Recommended max height of 40-50px

## Note:
The TrustedBy component will display these logos in grayscale by default, with color appearing on hover.
