# Team Images

Place your team member images in this directory.

## Required Images:
- member1.jpg
- member2.jpg
- member3.jpg
- member4.jpg

## Image Specifications:
- Recommended size: 600x800px (portrait orientation)
- Format: JPG, PNG, or WebP
- Aspect ratio: 3:4 works best with the card design

## Note:
The TeamSection component will automatically display these images in the carousel with the lime-green gradient overlay.
